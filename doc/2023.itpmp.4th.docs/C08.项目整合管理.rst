
项目整合管理
------------

　项目整合管理包括对隶属于项目管理过程组的各种过程和项目管理活动进行识别、定义、组合、统一和协调的各个过程。在项目管理中，整合兼具统一、合并、沟通和建立联系的性质，这些行动
应该贯穿项目始终。

管理基础
~~~~~~~~

+ 项目管理计划

   + 范围、需求、进度、成本、质量、资源、沟通、风险、采购、干系人管理计划

   + 变更、配置管理计划

   + 范围、进度、成本、绩效测量基准

   + 项目生命周期描述

   + 开发方法

+ 项目文件
