
项目管理概论
------------

项目经理的角色
~~~~~~~~~~~~~~

**项目经理的关键技能：** 项目管理、战略与商务、领导力。

**项目管理关键技能：** ① 重点关注并随之准备好所管理的各个项目的关键项目管理要素；② 针对每个项目裁剪，有选择的使用传统工具、敏捷工具、技术、犯法；③ 花时间制定完整计划并谨慎排定优先顺序；④ 管理项目制约因素包括进度、成本、资源风险等。

**领导力技能：** ① 人际交往；② 领导者品质和技能；③ 政策和权力。

价值驱动的项目管理知识体系
~~~~~~~~~~~~~~~~~~~~~~~~~~

**项目管理原则**

*项目管理原则* 包括：① 勤勉、尊重和关心他人；② 营造协作的项目团队环境；③ 促进干系人的有效参与；④ 聚焦于价值；⑤ 识别、评估和响应系统交互；⑥ 展现领导力行为；⑦ 根据环境进行裁剪；⑧ 将质量融入到过程和成果中；⑨ 驾驭复杂性；⑩ 优化风险应对；⑪ 拥抱适应性和韧性；⑫ 为实现目标而驱动变革。

**项目生命周期和项目阶段**

*开发生命周期：* 预测行（计划驱动型）、迭代型、增量型、适应型（敏捷型）和混合型

**项目管理知识领域**

.. include:: /doc/2023.itpmp.4th.tables/C06.概论.01.过程组与知识领域.rst

**助记：**
  * 启动：制定项目章程，识别干系人；2个过程
  * 规划：范围4，进度5，成本3，资源2，风险5，其他1
  * 执行：整合2个过程，有基准的领域无过程，资源管理3个过程，其他领域1个过程
  * 监控：实施整体变更控制，确认范围；12个过程
  * 结束：结束项目或阶段；1个过程。

**价值交付系统**

*价值交付系统* 描述了项目如何在系统内运作，为组织及其干系人创造价值。价值交付系统包括项目如何创造价值、价值交付组件和信息流。

*项目可以通过以下方式创造价值：* ① 创造满足客户或最终用户需要的新产品、服务或结果；② 做出积极的社会或环境贡献；③ 提高效率、生产力、效果或响应能力；④ 推动必要的变革，以促进组织向期望的未来状态过渡；⑤ 维持以前的项目集、项目或业务运营所带来的收益等。
